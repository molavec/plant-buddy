# Plant Buddy Docker Infraestructure

Docker containers to implement Plant Buddy.

* **EMQX:** MQTT Broker
* **Timescale:** Data collector
* **Mongo:** NoSQL Database

## Helpers services
* **mongo-express:** Mongo NoSql Client 
* **pgAdmin:** Postgre Client
